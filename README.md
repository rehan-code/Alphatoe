# Requirements
- C (std: c99)
  
# Usage
Run `make all` to compile all programs

## Alphatoe
`./alphatoe` 
Enables you to play a game of TicTacToe against the AI in the terminal

## Count Nodes
`./count_nodes`
Calculates the number of nodes and prints them

## Show Nodes
`./show_nodes {index}`
Displays the node at the specified index.
